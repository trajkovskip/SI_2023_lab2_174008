## 3. Цикломатска комплексност
Цикломатската комплексност на овој код е 9, истата ја добив преку формулата V(G) = P + 1, каде што P е бројот на предикатни јазли. Во случајов P=8, па цикломатската комплексност изнесува 9. Цикломатската комплексност, исто така се пресметува и како V(G) = R, каде R e бројот на региони, во овој случај _, и како V(G) = E - N + 2, каде Е е бројот на врски, а N се јазлите. Па цикломатската комплексност би била V(G) = _ - _ + 2 = _.
